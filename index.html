JavaScript Ma'lumotlar Turlari

Asosiy turlar va ularning ishlash usuli

Primitive Types

Number
Sonlar uchun
let age = 25;
let price = 99.99;
String
Matn uchun
let name = "Ahmad";
let msg = 'Salom';
Boolean
true/false qiymatlar
let isActive = true;
let isDone = false;
Undefined
Qiymat berilmagan
let x;
console.log(x); // undefined
Null
Bo'sh qiymat
let data = null;
Symbol
Noyob identifikator
let id = Symbol('id');

Object Types

Object
Kalit-qiymat juftliklari
let user = {
  name: "Ali",
  age: 20
};
Array
Ro'yxat ko'rinishida
let nums = [1, 2, 3];
let names = ["A", "B"];
Function
Bajariladigan kod
function hello() {
  return "Salom";
}
Date
Sana va vaqt
let now = new Date();
